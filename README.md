**TatamiCraft** adds traditional Japanese-style building blocks, Tatami/畳.

This mod introduces blocks such as **tatami mats and other Japanese architectural elements**, allowing players to create authentic Japanese interiors and buildings.

![Tatami blocks](https://cdn.modrinth.com/data/cached_images/bd47656a1e27f0b2e6e6d2832631e99e13634589.jpeg)

### Obtaining

Use commands like these.

```
/give @p tatamicraft:tatami
/give @p tatamicraft:tatami_half
```


### Crafting

Tatami can be crafted using wheat and string.

```
W W
S S
```

* W = Wheat
* S = String

You can also craft Half Tatami, or split a full Tatami into two halves.

Tatami blocks can be dyed into 10 or more different colors using standard Minecraft dyes.

### Features

* **Tatami and Japanese building blocks**
* **Thin block design** similar to carpets, ideal for floors and interior decoration
* Designed for **Japanese-style houses, temples, and traditional rooms**
* Lightweight and simple building-focused mod

### Compatibility

* **Minecraft:** 
  * 1.20.1
  * 1.21.11
* **Loader:**
  * Fabric
  * NeoForge

### v1.1.0 Update: Fusuma Blocks

Version 1.1.0 adds **Fusuma blocks**, traditional Japanese sliding doors used in tatami rooms and Japanese-style interiors.

Fusuma blocks are designed to match tatami floors and other Japanese architectural blocks, making it easier to create authentic washitsu rooms, temples, inns, and traditional houses.

These blocks can be used as decorative partitions or interior doors, helping players build more detailed Japanese-style spaces.
